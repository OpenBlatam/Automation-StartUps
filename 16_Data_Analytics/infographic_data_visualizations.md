## Design Specifications

### Color Palette
- **Primary Blue:** #1E40AF (main statistics)
- **Success Green:** #10B981 (positive metrics)
- **Warning Orange:** #F59E0B (attention-grabbing)
- **Neutral Gray:** #6B7280 (supporting text)
- **Background White:** #FFFFFF
- **Accent Purple:** #8B5CF6 (highlights)

### Typography
- **Headlines:** Bold, sans-serif (Montserrat Bold)
- **Body Text:** Clean, readable (Open Sans)
- **Numbers:** Extra bold, large size
- **Labels:** Medium weight, clear hierarchy

### Layout Guidelines
- **Maximum 5 data points per infographic**
- **Large, bold numbers for key statistics**
- **Consistent iconography throughout**
- **White space for visual breathing room**
- **Mobile-responsive design considerations**

### File Formats
- **High-res PNG:** 300 DPI for print
- **Web-optimized JPG:** 72 DPI for digital
- **Vector SVG:** Scalable for all sizes
- **PDF:** Complete infographic collection
