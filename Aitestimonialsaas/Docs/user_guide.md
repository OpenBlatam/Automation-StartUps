## Understanding Templates

### Template Psychology

Each template is designed based on psychological principles:

**Distinctive Qualities** ‚Üí **Differentiation Theory**
- Emphasizes unique features that competitors lack
- Appeals to customers seeking innovative solutions

**Recommendation** ‚Üí **Social Proof Theory**
- Leverages the power of peer recommendations
- Builds trust through third-party validation

**Specific Situation** ‚Üí **Concrete Evidence Theory**
- Provides tangible proof of value
- Makes abstract benefits more relatable

**Investment Worth** ‚Üí **Value Justification Theory**
- Addresses cost-benefit concerns
- Emphasizes ROI and long-term value

**Efficiency Improvement** ‚Üí **Productivity Theory**
- Focuses on time and resource savings
- Appeals to efficiency-minded buyers

### Template Selection Guide

| Use Case | Best Template | Why |
|----------|---------------|-----|
| Landing page hero section | Distinctive Qualities | Highlights what makes you unique |
| Product comparison page | Recommendation | Builds trust and credibility |
| Case study page | Specific Situation | Shows real-world application |
| Pricing page | Investment Worth | Justifies the cost |
| Feature page | Efficiency Improvement | Shows practical benefits |

## Advanced Features

### Batch Generation

Generate multiple testimonials at once:

1. Go to **Generate > Batch Mode**
2. Upload a CSV file or create multiple requests
3. Configure batch options:
   - **Parallel Processing**: Generate multiple testimonials simultaneously
   - **Model Rotation**: Use different AI models for variety
   - **Template Mix**: Combine different templates

**CSV Format**:
```csv
productName,industry,targetAudience,templateType,tone,length
"AI Platform","Technology","Developers","distinctiveQualities","technical","medium"
"CRM Software","Business","Sales Teams","recommendation","professional","short"
```

### Custom Prompts

Create your own testimonial prompts:

1. Go to **Templates > Custom**
2. Click **"Create Custom Template"**
3. Define your prompt with variables: `{product}`, `{benefit}`, `{audience}`
4. Test and save your template

**Example Custom Prompt**:
```
"As a {audience}, I've used {product} for {timeframe} and the {benefit} has been incredible. 
The {feature} feature specifically helped me {outcome}. I would definitely recommend this to other {audience}."
```

## Troubleshooting

### Common Issues

**Low Quality Scores**:
- **Problem**: Testimonials scoring below 70
- **Solution**: 
  - Add more specific details
  - Include emotional language
  - Ensure appropriate length
  - Use clear recommendations

**Generation Failures**:
- **Problem**: AI generation fails or times out
- **Solution**:
  - Check internet connection
  - Try different AI model
  - Reduce prompt complexity
  - Contact support if persistent

**Usage Limit Exceeded**:
- **Problem**: Hit monthly or daily limits
- **Solution**:
  - Upgrade your plan
  - Wait for limit reset
  - Optimize prompt efficiency
  - Use batch generation

**Poor Performance**:
- **Problem**: Low click-through or conversion rates
- **Solution**:
  - A/B test different templates
  - Improve testimonial quality
  - Better audience targeting
  - Update placement strategy
