# MATRIZ DE A/B TESTING PARA DMs

## Hipótesis por variable
- Versión (V1-V7): ciertos perfiles responden mejor a X enfoque
- Pregunta final: calificación vs curiosidad
- Métrica incluida: con métrica específica > sin métrica
- Tono: formal vs casual según industria
- Emoji: 0 vs 1 (LinkedIn)
- Timing: 10-12 vs 14-16 (Martes/Jueves)

## Diseño del experimento
| Experimento | Variable | A | B | Muestra mínima | KPI principal | Resultado |
|-------------|----------|---|---|----------------|---------------|-----------|
| EXP-01 | Versión | V2-ROI | V4-Autom | 50/contactos | Respuesta % | |
| EXP-02 | Pregunta | Calificación | Curiosidad | 50/contactos | Conversión a objetivo % | |
| EXP-03 | Métrica | Con métrica | Sin métrica | 50/contactos | Respuesta % | |
| EXP-04 | Tono | Formal | Casual | 50/contactos | Respuesta % | |
| EXP-05 | Emoji | 0 | 1 | 50/contactos | Respuesta % | |
| EXP-06 | Timing | 10-12 | 14-16 | 50/contactos | Respuesta % | |

## Reglas
- Un cambio por experimento
- Muestra mínima por variante ≥ 50 contactos
- Duración mínima: 2 semanas o 100 contactos/variante
- Controla por industria/rol al comparar

## Registro de resultados
- Anota aprendizajes en `SISTEMA_SEGUIMIENTO_DM.md`
- Actualiza DMs ganadores en los documentos principales
- Repite pruebas trimestralmente
