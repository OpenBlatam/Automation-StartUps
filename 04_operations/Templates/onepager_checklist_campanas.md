# Checklist 1 Página — Lanzamiento de Campaña

Pre‑Launch
- [ ] Brief aprobado (objetivos SMART, KPIs, presupuesto)
- [ ] Plan de medios y matriz de tests
- [ ] Creatividades cargadas y compliance OK
- [ ] Landings QA (perf, forms, a11y)
- [ ] Tracking GTM/GA4 + server‑side, UTMs verificados
- [ ] Dashboards y alertas activas
- [ ] Go/No‑Go firmado y on‑call asignado

Launch
- [ ] Soft launch (10–20%) activado
- [ ] Monitoreo T+1h/6h/24h y correcciones
- [ ] Changelog actualizado

Optimización
- [ ] Redistribución de budget hacia ganadores
- [ ] Iteración creativa/audiencias
- [ ] Revisión diaria + comité semanal

Cierre
- [ ] Reporte final y post‑mortem
- [ ] Librería de ganadores actualizada
- [ ] Archivado y KT
