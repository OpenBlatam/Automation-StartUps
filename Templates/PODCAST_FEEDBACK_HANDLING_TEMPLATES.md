### ğŸ¯ Template 1: Addressing Specific Segment Feedback

**Use Case:** When feedback targets a specific segment of your podcast

**Template:**
```
Handling podcast feedback

Our most recent episode garnered feedback that was {ADJECTIVE} towards the {TOPIC} segment. In order to address these concerns and enhance the overall {ADJECTIVE} quality of our podcast, what steps can we take to {VERB} this particular segment?
```

**Placeholder Options:**

**{ADJECTIVE} - Feedback Tone:**
- Negative: "critical", "negative", "disappointing", "concerning", "unfavorable"
- Mixed: "mixed", "divided", "controversial", "polarized"
- Positive: "positive", "appreciative", "enthusiastic", "favorable"
- Neutral: "neutral", "indifferent", "moderate"

**{TOPIC} - Segment Types:**
- Content: "interview", "intro", "outro", "Q&A", "discussion", "analysis", "storytelling"
- Format: "opening", "closing", "transitions", "ad breaks", "sponsor segments"
- Technical: "audio quality", "editing", "music", "sound effects"

**{VERB} - Action Verbs:**
- Improve: "improve", "enhance", "refine", "polish", "upgrade"
- Restructure: "restructure", "reorganize", "redesign", "reformat"
- Optimize: "optimize", "streamline", "perfect", "maximize"

**Examples:**

*Example 1 - Critical Feedback:*
```
Our most recent episode garnered feedback that was critical towards the interview segment. In order to address these concerns and enhance the overall professional quality of our podcast, what steps can we take to improve this particular segment?
```

*Example 2 - Mixed Feedback:*
```
Our most recent episode garnered feedback that was mixed towards the Q&A segment. In order to address these concerns and enhance the overall engaging quality of our podcast, what steps can we take to restructure this particular segment?
```

*Example 3 - Technical Issue:*
```
Our most recent episode garnered feedback that was negative towards the audio quality segment. In order to address these concerns and enhance the overall production quality of our podcast, what steps can we take to optimize this particular segment?
```

**Expected Output:**
- Specific actionable steps
- Prioritized recommendations
- Implementation timeline
- Resource requirements
- Success metrics

#!/usr/bin/env python3
"""
Analizador de Feedback de Podcast
Categoriza y analiza feedback automÃ¡ticamente
"""

import re
from collections import Counter
from datetime import datetime

class PodcastFeedbackAnalyzer:
    def __init__(self):
        self.categories = {
            'audio': ['audio', 'sound', 'quality', 'mic', 'microphone', 'volume', 'noise'],
            'content': ['content', 'topic', 'subject', 'information', 'depth', 'detail'],
            'pacing': ['pacing', 'speed', 'rushed', 'slow', 'fast', 'rhythm'],
            'format': ['format', 'structure', 'length', 'episode', 'segment'],
            'engagement': ['interactive', 'engagement', 'questions', 'participation', 'community']
        }
        
        self.sentiment_keywords = {
            'positive': ['great', 'excellent', 'love', 'amazing', 'fantastic', 'wonderful', 'good'],
            'negative': ['poor', 'bad', 'terrible', 'awful', 'hate', 'disappointing', 'worst'],
            'neutral': ['okay', 'fine', 'average', 'decent', 'standard']
        }
    
    def categorize_feedback(self, feedback_text):
        """Categoriza el feedback en categorÃ­as principales"""
        feedback_lower = feedback_text.lower()
        category_scores = {}
        
        for category, keywords in self.categories.items():
            score = sum(1 for keyword in keywords if keyword in feedback_lower)
            if score > 0:
                category_scores[category] = score
        
        if category_scores:
            return max(category_scores, key=category_scores.get)
        return 'general'
    
    def analyze_sentiment(self, feedback_text):
        """Analiza el sentimiento del feedback"""
        feedback_lower = feedback_text.lower()
        sentiment_scores = {'positive': 0, 'negative': 0, 'neutral': 0}
        
        for sentiment, keywords in self.sentiment_keywords.items():
            sentiment_scores[sentiment] = sum(1 for keyword in keywords if keyword in feedback_lower)
        
        if max(sentiment_scores.values()) == 0:
            return 'neutral'
        
        return max(sentiment_scores, key=sentiment_scores.get)
    
    def suggest_template(self, category, sentiment):
        """Sugiere el template apropiado basado en categorÃ­a y sentimiento"""
        template_map = {
            'audio': 1 if sentiment == 'negative' else 3,
            'content': 3,
            'pacing': 4,
            'format': 4,
            'engagement': 5
        }
        
        return template_map.get(category, 3)
    
    def analyze_feedback_batch(self, feedback_list):
        """Analiza un lote de feedback"""
        results = {
            'total': len(feedback_list),
            'by_category': Counter(),
            'by_sentiment': Counter(),
            'template_suggestions': Counter(),
            'detailed': []
        }
        
        for feedback in feedback_list:
            category = self.categorize_feedback(feedback['text'])
            sentiment = self.analyze_sentiment(feedback['text'])
            template = self.suggest_template(category, sentiment)
            
            results['by_category'][category] += 1
            results['by_sentiment'][sentiment] += 1
            results['template_suggestions'][template] += 1
            
            results['detailed'].append({
                'text': feedback['text'],
                'category': category,
                'sentiment': sentiment,
                'template': template,
                'date': feedback.get('date', datetime.now().isoformat())
            })
        
        return results

# Ejemplo de uso

#!/usr/bin/env python3
"""
AnÃ¡lisis de Tendencias de Feedback
Identifica patrones y tendencias a lo largo del tiempo
"""

import pandas as pd
from datetime import datetime, timedelta
from collections import defaultdict
import matplotlib.pyplot as plt

class FeedbackTrendAnalyzer:
    def __init__(self, feedback_data):
        self.df = pd.DataFrame(feedback_data)
        self.df['date'] = pd.to_datetime(self.df['date'])
    
    def analyze_trends(self, period='weekly'):
        """Analiza tendencias por perÃ­odo"""
        if period == 'weekly':
            self.df['period'] = self.df['date'].dt.to_period('W')
        elif period == 'monthly':
            self.df['period'] = self.df['date'].dt.to_period('M')
        else:
            self.df['period'] = self.df['date'].dt.date
        
        trends = {
            'volume': self.df.groupby('period').size().to_dict(),
            'sentiment': self.df.groupby('period')['sentiment'].apply(
                lambda x: (x == 'positive').sum() / len(x) * 100
            ).to_dict(),
            'categories': self.df.groupby(['period', 'category']).size().to_dict(),
            'priority_distribution': self.df.groupby(['period', 'priority']).size().to_dict()
        }
        
        return trends
    
    def identify_emerging_issues(self, threshold=3):
        """Identifica problemas emergentes"""
        recent_feedback = self.df[
            self.df['date'] >= datetime.now() - timedelta(days=7)
        ]
        
        category_counts = recent_feedback['category'].value_counts()
        emerging = category_counts[category_counts >= threshold]
        
        return emerging.to_dict()
    
    def predict_future_feedback(self, days=30):
        """Predice volumen de feedback futuro"""
        # Simple linear regression
        volume_by_date = self.df.groupby(self.df['date'].dt.date).size()
        
        # Calcular promedio y tendencia
        avg_volume = volume_by_date.mean()
        trend = (volume_by_date.iloc[-7:].mean() - 
                volume_by_date.iloc[-14:-7].mean())
        
        predicted = avg_volume + (trend * (days / 7))
        
        return {
            'predicted_volume': int(predicted),
            'confidence': 'medium',
            'based_on': len(volume_by_date)
        }
    
    def generate_trend_report(self):
        """Genera reporte completo de tendencias"""
        trends = self.analyze_trends('weekly')
        emerging = self.identify_emerging_issues()
        prediction = self.predict_future_feedback()
        
        report = f"""
# Trend Analysis Report
Generated: {datetime.now().strftime('%Y-%m-%d %H:%M')}

## Volume Trends
{self._format_volume_trends(trends['volume'])}

## Sentiment Trends
{self._format_sentiment_trends(trends['sentiment'])}

## Emerging Issues
{self._format_emerging_issues(emerging)}

## Predictions
- Predicted volume (next 30 days): {prediction['predicted_volume']}
- Confidence: {prediction['confidence']}
- Based on: {prediction['based_on']} days of data
        """
        
        return report
    
    def _format_volume_trends(self, volume_dict):
        """Formatea tendencias de volumen"""
        lines = []
        for period, count in sorted(volume_dict.items()):
            lines.append(f"- {period}: {count} feedback items")
        return "\n".join(lines)
    
    def _format_sentiment_trends(self, sentiment_dict):
        """Formatea tendencias de sentimiento"""
        lines = []
        for period, percentage in sorted(sentiment_dict.items()):
            lines.append(f"- {period}: {percentage:.1f}% positive")
        return "\n".join(lines)
    
    def _format_emerging_issues(self, emerging_dict):
        """Formatea problemas emergentes"""
        if not emerging_dict:
            return "No emerging issues detected"
        
        lines = []
        for category, count in sorted(emerging_dict.items(), 
                                     key=lambda x: x[1], 
                                     reverse=True):
            lines.append(f"- {category}: {count} mentions (needs attention)")
        return "\n".join(lines)

import re
from collections import Counter

class EmotionAnalyzer:
    def __init__(self):
        self.emotions = {
            'joy': ['love', 'amazing', 'fantastic', 'excited', 'great', 'wonderful', 'awesome'],
            'gratitude': ['thanks', 'thank you', 'appreciate', 'grateful'],
            'enthusiasm': ["can't wait", 'looking forward', 'excited', 'eager'],
            'satisfaction': ['satisfied', 'happy', 'pleased', 'content'],
            'frustration': ['frustrated', 'annoyed', 'disappointed', 'irritated'],
            'anger': ['angry', 'mad', 'furious', 'hate', 'terrible'],
            'sadness': ['sad', 'disappointed', 'unhappy', 'depressed'],
            'worry': ['worried', 'concerned', 'anxious', 'nervous'],
            'curiosity': ['wondering', 'curious', 'interested', 'question'],
            'indifference': ['okay', 'fine', 'whatever', 'meh'],
            'confusion': ['confused', 'unclear', "don't understand", 'puzzled']
        }
    
    def analyze_emotions(self, text):
        """Analiza emociones en el texto"""
        text_lower = text.lower()
        emotion_scores = Counter()
        
        for emotion, keywords in self.emotions.items():
            for keyword in keywords:
                # Buscar palabra completa
                pattern = r'\b' + re.escape(keyword) + r'\b'
                matches = len(re.findall(pattern, text_lower))
                emotion_scores[emotion] += matches
        
        # Normalizar scores
        total = sum(emotion_scores.values())
        if total > 0:
            emotion_percentages = {
                emotion: (count / total) * 100 
                for emotion, count in emotion_scores.items()
            }
        else:
            emotion_percentages = {}
        
        # Determinar emociÃ³n dominante
        dominant_emotion = emotion_scores.most_common(1)
        dominant = dominant_emotion[0][0] if dominant_emotion else 'neutral'
        
        return {
            'dominant_emotion': dominant,
            'emotion_scores': dict(emotion_scores),
            'emotion_percentages': emotion_percentages,
            'confidence': max(emotion_percentages.values()) if emotion_percentages else 0
        }
    
    def get_emotion_category(self, emotion):
        """Categoriza emociÃ³n como positiva, negativa o neutra"""
        positive = ['joy', 'gratitude', 'enthusiasm', 'satisfaction']
        negative = ['frustration', 'anger', 'sadness', 'worry']
        neutral = ['curiosity', 'indifference', 'confusion']
        
        if emotion in positive:
            return 'positive'
        elif emotion in negative:
            return 'negative'
        else:
            return 'neutral'

# Ejemplo de uso

    print(f"Dominant emotion: {result['dominant_emotion']}")
    print(f"Category: {analyzer.get_emotion_category(result['dominant_emotion'])}")
    print(f"Confidence: {result['confidence']:.1f}%")

import numpy as np
from sklearn.linear_model import LinearRegression
from datetime import datetime, timedelta

class FeedbackVolumePredictor:
    def __init__(self, historical_data):
        self.data = historical_data
        self.model = LinearRegression()
    
    def prepare_data(self, days_back=90):
        """Prepara datos para entrenamiento"""
        end_date = datetime.now()
        start_date = end_date - timedelta(days=days_back)
        
        # Agrupar por dÃ­a
        daily_volumes = {}
        for item in self.data:
            date = datetime.fromisoformat(item['date']).date()
            if start_date.date() <= date <= end_date.date():
                daily_volumes[date] = daily_volumes.get(date, 0) + 1
        
        # Convertir a arrays
        dates = sorted(daily_volumes.keys())
        X = np.array([(d - dates[0]).days for d in dates]).reshape(-1, 1)
        y = np.array([daily_volumes[d] for d in dates])
        
        return X, y, dates
    
    def train(self):
        """Entrena el modelo"""
        X, y, _ = self.prepare_data()
        self.model.fit(X, y)
        return self.model.score(X, y)
    
    def predict(self, days_ahead=30):
        """Predice volumen futuro"""
        X, y, dates = self.prepare_data()
        self.model.fit(X, y)
        
        # Predecir dÃ­as futuros
        last_date = dates[-1]
        future_dates = [
            last_date + timedelta(days=i+1) 
            for i in range(days_ahead)
        ]
        future_X = np.array([
            (d - dates[0]).days 
            for d in future_dates
        ]).reshape(-1, 1)
        
        predictions = self.model.predict(future_X)
        
        return {
            'dates': future_dates,
            'predicted_volumes': predictions.tolist(),
            'average': float(np.mean(predictions)),
            'confidence': 'medium'  # Basado en RÂ² score
        }
    
    def identify_anomalies(self, threshold=2):
        """Identifica dÃ­as con volumen anormal"""
        X, y, dates = self.prepare_data()
        self.model.fit(X, y)
        
        predictions = self.model.predict(X)
        residuals = y - predictions
        std = np.std(residuals)
        
        anomalies = []
        for i, (date, actual, predicted) in enumerate(zip(dates, y, predictions)):
            if abs(residuals[i]) > threshold * std:
                anomalies.append({
                    'date': date,
                    'actual': int(actual),
                    'predicted': float(predicted),
                    'deviation': float(residuals[i])
                })
        
        return anomalies

class FeedbackImpactCalculator:
    def __init__(self):
        self.metrics_weights = {
            'completion_rate': 0.3,
            'sentiment_score': 0.25,
            'subscriber_growth': 0.2,
            'engagement': 0.15,
            'retention': 0.1
        }
    
    def calculate_impact(self, before_metrics, after_metrics):
        """Calcula impacto total de mejoras"""
        impact_scores = {}
        
        for metric, weight in self.metrics_weights.items():
            if metric in before_metrics and metric in after_metrics:
                before = before_metrics[metric]
                after = after_metrics[metric]
                
                if before > 0:
                    improvement = ((after - before) / before) * 100
                    impact_scores[metric] = {
                        'improvement': improvement,
                        'weighted_score': improvement * weight
                    }
        
        total_impact = sum(
            score['weighted_score'] 
            for score in impact_scores.values()
        )
        
        return {
            'total_impact': total_impact,
            'impact_by_metric': impact_scores,
            'grade': self._calculate_grade(total_impact)
        }
    
    def _calculate_grade(self, impact):
        """Calcula calificaciÃ³n basada en impacto"""
        if impact >= 50:
            return 'A+'
        elif impact >= 40:
            return 'A'
        elif impact >= 30:
            return 'B+'
        elif impact >= 20:
            return 'B'
        elif impact >= 10:
            return 'C+'
        else:
            return 'C'
    
    def generate_impact_report(self, before, after, improvement_name):
        """Genera reporte de impacto"""
        impact = self.calculate_impact(before, after)
        
        report = f"""
# Impact Report: {improvement_name}

## Overall Impact
**Score:** {impact['total_impact']:.1f}%
**Grade:** {impact['grade']}

## Impact by Metric
"""
        for metric, data in impact['impact_by_metric'].items():
            report += f"""
### {metric.replace('_', ' ').title()}
- Improvement: {data['improvement']:.1f}%
- Weighted Score: {data['weighted_score']:.1f}%
"""
        
        return report
