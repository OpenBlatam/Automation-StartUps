### Scripts SQL para Procesamiento

**Query: Eficiencia por Usuario**
```sql
SELECT 
    u.id,
    u.email,
    COUNT(DISTINCT bp.batch_id) as batches_processed,
    SUM(bp.document_count) as total_documents,
    AVG(bp.quality) as avg_quality,
    SUM(sp.amount_saved) as total_savings,
    SUM(bp.points) as total_points
FROM users u
JOIN batch_points bp ON u.id = bp.user_id
LEFT JOIN savings_points sp ON u.id = sp.user_id
WHERE bp.timestamp >= CURRENT_DATE - INTERVAL '30 days'
GROUP BY u.id, u.email
ORDER BY total_points DESC;
```

**Query: Impacto de Templates Personalizados**
```sql
SELECT 
    r.reward_type,
    COUNT(rd.id) as redemptions,
    AVG(rd.quality_improvement) as avg_quality_improvement,
    AVG(rd.time_saved_percent) as avg_time_saved,
    SUM(rd.cost_saved) as total_cost_saved
FROM bulk_rewards r
JOIN redemptions rd ON r.id = rd.reward_id
WHERE r.reward_type = 'custom_template'
GROUP BY r.reward_type;
```
