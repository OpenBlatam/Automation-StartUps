# Variantes de Encuestas Post-Webinar

## Variante 1: Encuesta Básica de Satisfacción
```json
{
  "encuesta": {
    "titulo": "¿Cómo Fue tu Experiencia?",
    "preguntas": [
      "¿Qué tan útil fue el contenido del webinar? (1-10)",
      "¿Recomendarías este webinar a otros? (Sí/No)",
      "¿Qué te gustó más del webinar?",
      "¿Qué mejorarías?"
    ],
    "boton": "Enviar Opinión"
  }
}
```
