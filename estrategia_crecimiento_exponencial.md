## 1. MODELO DE CRECIMIENTO EXPONENCIAL

### Fórmula de Crecimiento Viral

#### Viral Coefficient (K)
- **K = 1.5**: Cada usuario trae 1.5 usuarios nuevos
- **K = 2.0**: Crecimiento exponencial sostenido
- **K = 3.0**: Crecimiento explosivo
- **K = 5.0**: Crecimiento viral masivo

#### Cálculo de Crecimiento
```
Usuarios(t) = Usuarios(0) × (1 + K)^t

Donde:
- Usuarios(0) = 1,000 usuarios iniciales
- K = 2.0 (viral coefficient)
- t = tiempo en meses

Resultado:
- Mes 1: 3,000 usuarios
- Mes 2: 9,000 usuarios
- Mes 3: 27,000 usuarios
- Mes 6: 729,000 usuarios
- Mes 12: 531,441,000 usuarios
```
