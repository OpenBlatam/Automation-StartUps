# AI ROI Calculator Template

## Calculadora de Retorno de Inversión para Productos de IA

### Fórmula Base de ROI
```
ROI = (Beneficios Anuales - Costos Anuales) / Costos Anuales × 100
```

## Herramientas de Cálculo

### Excel Template
```
=((Beneficios_Anuales - Costos_Anuales) / Costos_Anuales) * 100
```

### Google Sheets Formula
```
=ROUND(((B2-C2)/C2)*100,2)
```

### JavaScript Calculator
```javascript
function calculateROI(benefits, costs) {
    return ((benefits - costs) / costs) * 100;
}
```
