## üöÄ **Usage Examples**

### **Basic AI Predictions**
```python
from ai_ml_engine import get_ai_engine

ai_engine = get_ai_engine()
launch_data = {
    "budget": 250000,
    "team_size": 12,
    "market_size": 5000000,
    "competition_level": 0.7
}

# Predict success probability
success_pred = ai_engine.predict_success_probability(launch_data)
print(f"Success Probability: {success_pred.prediction:.1%}")

# Optimize budget allocation
budget_opt = ai_engine.optimize_budget_allocation(launch_data)
print(f"Optimal Budget: {budget_opt}")
```

### **Predictive Analytics**
```python
from predictive_analytics import get_predictive_analytics

analytics = get_predictive_analytics()

# Add time series data
analytics.add_metric_data("success_rate", 0.85)
analytics.add_metric_data("budget_utilization", 0.65)

# Generate forecasts
forecasts = analytics.generate_forecasts(["success_rate", "budget_utilization"])
for metric, forecast in forecasts.items():
    print(f"{metric}: {forecast.predicted_value:.3f}")
```

### **Intelligent Automation**
```python
from intelligent_automation import get_automation_engine

automation = get_automation_engine()

# Create automation rules
rule_ids = automation.create_launch_automation_rules("launch_001")

# Execute automation
execution_id = automation.execute_rule(rule_ids[0])
print(f"Automation executed: {execution_id}")
```

### **Advanced Monitoring**
