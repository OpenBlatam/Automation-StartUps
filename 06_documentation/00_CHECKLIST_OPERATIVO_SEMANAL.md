# ✅ Checklist Operativo Semanal (Innovación en Marcha)

## Antes de Enviar
- [ ] Campos CRM listos: `dm_variant`, `lead_score`, `primary_objection`, `best_send_hour`, `channel`, `package_tier`.
- [ ] Prompts/plantillas versionados (tag semana).
- [ ] Stop-list y límites de frecuencia activos por canal.

## Lunes (Plan)
- [ ] Revisar dashboard (variante/canal/horario/industria).
- [ ] Seleccionar 2 hipótesis A/B y definir cambios mínimos.
- [ ] Publicar changelog de prompts.

## Miércoles (Calidad y Pipeline)
- [ ] Auditoría de marca: muestra del 10% (tono, CTA, datos correctos).
- [ ] Revisar objeciones top y actualizar respuestas/plantillas.
- [ ] Revisar tiempos de respuesta y reprogramar horarios si cae performance.

## Viernes (Aprendizaje y Rollout)
- [ ] Documentar resultados A/B y decisiones (duplicar/pausar/testear).
- [ ] Actualizar librería de variantes y prompts.
- [ ] Preparar informe 1 página (KPIs, decisiones, próximos tests).

## Mensual
- [ ] Revisión de pricing/membresías según uso/ROI.
- [ ] Reporte de partners: ranking, payouts, calidad.
- [ ] Auditoría de privacidad y consentimiento.

## Métricas Umbral (alarma)
- [ ] Reply < 15% por 2 semanas → revisar hooks/hora/canal.
- [ ] DM→Demo < 5% → revisar CTA/oferta/segmentación.
- [ ] Demo→Win < 20% → revisar propuesta/ROI/fit.
- [ ] Churn > 8% → revisar packaging/soporte/valor percibido.
