# Plantillas de Respuestas DM (Objeciones + BANT)

Respuestas base para objeciones frecuentes y calificación rápida (BANT).

---

## Objeciones Frecuentes

### "Ya tenemos algo similar"
```
Entiendo, [NOMBRE]. [HERRAMIENTA] es sólida.
La diferencia de [NUESTRO_PRODUCTO] está en [DIFERENCIADOR] (ej: integración/flujo/ROI).
¿Te muestro en 10 min si suma a lo que ya usan? Sin compromiso.
```

### "No hay presupuesto"
```
Totalmente válido. ¿Cuándo revisan presupuesto normalmente?
Si te sirve mientras tanto, te comparto [RECURSO] y vemos en [MES/FECHA].
```

### "Probamos IA y no funcionó"
```
He visto eso. Suele pasar por dataset/prompts/flujo.
¿Puedes contarme qué falló antes? Con eso te muestro cómo lo abordamos distinto.
```

### "Estamos en otra prioridad ahora"
```
Entiendo. ¿Te parece si te dejo un recurso breve y retomamos en [X semanas]?
Si surge algo antes, aquí estoy.
```

---

## Respuestas Aceleradoras

### "Envíame info"
```
Te envío 2 cosas concretas: 1) caso 1 página con métricas, 2) demo de 10 min.
¿Prefieres ver primero el caso o la demo?
```

### "Llámame"
```
Genial. ¿Qué día/hora te funciona? Opciones: [OPCION_1] / [OPCION_2].
También te paso un link de calendario si prefieres.
```

---

## BANT Rápido (3 preguntas)

```
Para no hacerte perder tiempo, 3 rápidas:
1) ¿Cuál es el principal desafío con [AREA] hoy?
2) Si lo resuelves, ¿qué impacto tendría (métrica/tiempo)?
3) ¿Hay una ventana ideal para implementarlo (mes)?
```

---

## Notas de Uso
- Adapta el tono a canal y perfil.
- Responde con valor específico (caso/métrica/flujo).
- Propón el siguiente paso claro (demo/llamada/opciones).
