# Escalera de Valor: Curso IA + SaaS IA Marketing + IA Bulk Documentos

Objetivo: Definir 4 pasos (Lead Magnet, Tripwire, Core, High-Ticket) por cada oferta, con automatización (Notion/Airtable + Zapier/Make + CRM), KPIs y SOPs.

---

0) Arquitectura transversal
- Base: Notion o Airtable con tablas: Leads, Eventos, Ofertas, Tareas, Webinars, Demos, Compras.
- Orquestación: Zapier/Make (formularios, emails, recordatorios, segmentación por score).
- CRM: HubSpot/Salesforce (o vistas en Airtable/Notion) con etapas: Identificado, Contactado, Demo, Propuesta, Cerrado.
- Identidad: email + LinkedIn + UTM + source_variant.

---

1) Curso de IA + Webinars

Paso 1: Lead Magnet (Gratis)
- Promesa: Guía micro-learning: 1 webinar -> 7 micro-lecciones (+18-30% completion).
- Activos: PDF + plantilla Notion + checklist pre-webinar.
- Automatización: Form -> crear Lead + Evento descarga -> Email entrega -> Email caso -> Invitación webinar.
- KPIs: CVR > 35%, Open E0 > 55%, CTR > 12%.

Paso 2: Tripwire ($7-$47)
- Oferta: Mini-curso + toolkit para convertir un webinar en 12 activos.
- Upsell: Webinar premium o bundle templates.
- Automatización: Compra -> onboarding -> invitación a Core.
- KPIs: CR checkout > 5-10%, upsell > 20%.

Paso 3: Core ($197-$497)
- Promesa: Curso completo para levantar un sistema de recomendaciones en 4-6 semanas.
- Entregables: currículo, plantillas, casos, soporte grupal 4 semanas.
- Automatización: segmentación por asistencia; deadline + recordatorios; si 50% progreso -> invitar a High-Ticket.
- KPIs: LM->Core 2-4%, asistencia > 30%, completion > 45%.

Paso 4: High-Ticket ($1,500+)
- Oferta: Implementación guiada de micro-learning y monetización de webinars.
- Automatización: señales de intent (curso completado, empresa >5) -> tarea SDR + secuencia consultiva.
- KPIs: invitación->llamada > 25%, cierre 15-30%.

---

2) SaaS IA aplicado al marketing (creative testing/ROAS)

Paso 1: Lead Magnet (Gratis)
- Promesa: Creative Audit Express (mapa de fatiga + 3 hipótesis).
- Automatización: Form -> generar audit -> Email entrega + CTA demo.
- KPIs: CVR > 30%, demo aceptada > 12%.

Paso 2: Tripwire ($7-$47)
- Oferta: Pack de prompts/plantillas (20 variaciones por plataforma) + guía de test.
- Automatización: Compra -> entrega + invitación demo 20 min.
- KPIs: CR checkout > 5-8%, demo booking > 15%.

Paso 3: Core ($197-$497)
- Oferta: Piloto 2 semanas con cuenta real (predicción de fatiga + rotación + 3 hipótesis).
- Automatización: Checklist implementación + alertas diarias.
- KPIs: Demo->Piloto > 25%, Piloto->Plan mensual > 30%.

Paso 4: High-Ticket ($1,500+/mes)
- Oferta: Plan mensual del SaaS + acompañamiento.
- KPIs: Payback < 2 meses, retención 3+ meses, ROAS +12-25% en 4-8 semanas.

---

3) IA Bulk Documentos (1 consulta -> 3 docs: Propuesta, Brief, SOP)

Paso 1: Lead Magnet (Gratis)
- Promesa: Sandbox demo en <24h con 3 documentos generados.
- Automatización: Form -> solicitar 1 doc ejemplo -> sandbox -> envío + CTA demo.
- KPIs: CVR > 35%, sandbox aceptado > 25%.

Paso 2: Tripwire ($7-$47)
- Oferta: Kit de templates inteligentes por rol.
- Automatización: Entrega + invitación a piloto.
- KPIs: CR checkout > 6-10%.

Paso 3: Core ($197-$497)
- Oferta: Piloto 2 semanas (6-12 documentos reales + dashboard de aprobaciones).
- Automatización: pipeline por documento + notificaciones.
- KPIs: Demo->Piloto > 30%, ahorro tiempo > 70%.

Paso 4: High-Ticket ($1,500+)
- Oferta: Implementación completa + training + governance.
- KPIs: Throughput 5-10x, time-to-proposal < 24h, retención 3+ meses.

---

Automatizaciones (resumen de recetas)
- Form submit -> Upsert Lead -> Evento -> Email 0 -> follow-up 48h.
- Registro webinar -> reminders T-24h/T-2h -> replay -> segmentación por asistencia.
- Compra Tripwire -> cross-sell Core (ventana 72h).
- Booking demo -> checklist pre-demo -> post-demo propuesta.

Vistas operativas (Notion/Airtable)
- Hoy: leads con tareas.
- Post-webinar: no asistentes.
- High-intent: score >= 6 con evento reciente.

KPIs globales
- TOFU: CVR LM >= 30-40%.
- MOFU: asistencia webinar >= 30-35%, aceptación demo >= 12-20%.
- BOFU: CR Tripwire >= 5-10%, CR Core >= 2-4%, CR High-Ticket >= 15-30%.
- Valor: AOV, LTV 90/180d, Payback.

SOPs clave
- Pre-webinar: agenda, emails T-48/T-24/T-2, página replay, encuesta.
- Post-webinar: replay T+1h/T+24h, oferta Core, deadline, segmentación.
- Pre-demo: accesos/plantillas, hipótesis, agenda, grabación.
- Post-demo: resumen, propuesta, fechas, responsables.

  - High-Ticket: $1,500 sprint 2 semanas / $2,500 4 semanas

  - Core (Piloto): $297 setup único o $497 con 2 semanas soporte
  - High-Ticket: $1,500–$4,000/mes según gasto en ads e integraciones
- IA Bulk Documentos:
  - Tripwire: $27 kit templates inteligentes
  - Core (Piloto): $297 (6 docs) / $497 (12 docs)
  - High-Ticket: $1,500–$3,000 implementación + $500–$1,500/mes soporte

Anexo B) Secuencias de emails/DMs por paso
- Lead Magnet → Entrega (Día 0) → Caso (Día 2) → Invitación (Día 5)
- Tripwire → Onboarding (Día 0) → ROI específico (Día 1) → Oferta Core (Día 3)
- Core → Deadline 1 (Día 0) → Caso sector (Día 2) → Extensión oculta (Día 4)
- High-Ticket → Email consultivo (Día 0) → DM con horarios (Día 1) → Llamada 10 min (Día 3)
- DMs: usar variantes de los documentos 01/02/03 según producto y situación.

Anexo C) Esquema Notion/Airtable (campos)
- Leads: id, nombre, email, linkedin_url, empresa, rol, país, fuente, utm_source, utm_campaign, score, owner, etapa
- Eventos: id, lead_id, tipo, producto, metadata_json, timestamp
- Ofertas: id, producto, escalón, precio, url_checkout, estado
- Compras: id, lead_id, oferta_id, monto, fecha, cupón, método
- Tareas: id, lead_id, tipo, due_date, estado, owner
- Webinars: id, titulo, fecha, url_registro, registrados, asistentes, ventas
- Demos: id, lead_id, fecha, resultado, notas, siguiente_paso

Anexo D) Recetas Zapier/Make (pseudoflujos)
- New Form (LM) → Upsert Lead → Evento:descarga → Send Email 0 → Create Task follow-up 48h
- New Webinar Registration → Delay T-24h/T-2h → Send Reminders → Post Replay → Branch attended vs no-show
- New Purchase (Tripwire) → Tag producto → Send Onboarding → Offer Core (72h window)
- New Calendly Booking (Demo) → Create Task checklist → Post-call: Send Proposal → Update etapa CRM

Anexo E) Dashboards/KPIs y UTM
- KPIs por etapa: CVR LM, Open/CTR, Asistencia, Demo rate, CR Tripwire/Core/HT, AOV, LTV, Payback
- Gráficas: funnel por producto, heatmap por variante, cohortes 30/60/90 días
- UTM taxonomy: utm_source (li,email,wa,ads), utm_medium (organic,paid,dm), utm_campaign (producto_escalon_variante_vX)

Anexo F) Mecánicas de oferta
- Order bump (Tripwire checkout): checklist premium (+$9)
- Upsell (post‑compra Tripwire): bundle templates/mini‑curso (+$17)
- Downsell (si rechaza Core): plan de 2 pagos o versión lite ($197)
- Garantías: 7 días para Tripwire/Core, piloto SaaS/IA Bulk “si no mejora métrica X, no se cobra”

Anexo G) Gobernanza (RACI) y SLA
- RACI: Growth (dueño funnels), Producto (activos/entregables), Ventas (demos/HT), Ops (automatizaciones/CRM)
- SLA: respuesta lead < 24h, entrega sandbox/audit < 24h, propuesta post‑demo < 24h

Anexo H) Plan 90 días
- Mes 1: fundaciones (3 landings LM, secuencias, dashboards)
- Mes 2: optimización (A/B pricing tripwire, variantes DM, recordatorios)
- Mes 3: escalamiento (ads ligeros, partnerships, automatización avanzada)
