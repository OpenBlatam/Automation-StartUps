# Investor Deck Template: Slide-by-Slide Guide

## Slide 1: Title Slide
**Template Content:**
```
[Company Logo]
[Company Name]
[Tagline: One sentence value proposition]

[Founder Names & Titles]
[Contact Information]
[Date]
```

**ClickUp Brain Links:**
- Company branding assets
- Founder bios and LinkedIn profiles
- Contact information document

---

## Slide 2: Problem Statement
**Template Content:**
```
The Problem We're Solving

[Large, bold problem statement]

• [Specific pain point 1 with statistic]
• [Specific pain point 2 with statistic]  
• [Specific pain point 3 with statistic]

[Visual: Problem illustration or customer quote]
```

**Team Alignment Questions:**
- What is the most painful problem our customers face?
- How much does this problem cost them annually?
- Why hasn't this been solved effectively before?

## Presentation Tips

### Timing Guidelines
- Total presentation: 15-20 minutes
- Q&A: 20-30 minutes
- Key slides: 1-2 minutes each
- Demo: 3-5 minutes

### Visual Best Practices
- Use consistent color scheme and fonts
- Include charts and graphs for data
- Keep text minimal and readable
- Use high-quality images and screenshots

### Delivery Tips
- Practice transitions between slides
- Prepare for common investor questions
- Have backup slides ready for deep dives
- Follow up within 24 hours with requested materials
