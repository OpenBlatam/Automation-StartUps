# Playbooks de Llamadas (Discovery / Cierre)

Estructura general (15–20 min)
- 2 min: contexto + objetivo
- 8–10 min: discovery (preguntas clave)
- 3–5 min: demo/resumen específico
- 2–3 min: siguientes pasos (claro y con fecha)

Curso IA / Webinars
- Discovery
  - ¿Cuántos webinars/mes? ¿Completion actual? ¿Qué bloquea?
  - ¿Tienen backlog de webinars? ¿Equipo/timing?
- Cierre
  - “Según lo que dices, Core te lleva de 0 a sistema en 4–6 semanas; propongo empezar este lunes. ¿Prefieres plan completo o lite?”
- Objeciones
  - “No tiempo” → “Setup 15 min; resto automatizado + replay.”
  - “Ya tenemos algo” → “Integra y amplifica; mostramos comparativa.”

SaaS IA Marketing (ROAS)
- Discovery
  - Presupuesto mensual, ROAS actual, % creatividades >14 días
  - Plataformas activas, equipo, cadencia de tests
- Cierre
  - “Piloto 2 semanas: predicción de fatiga + rotación; si no mejora CTR, no se cobra. ¿Agendamos kickoff?”
- Objeciones
  - “Sin tiempo” → “Automatiza testing; tú revisas resultados.”
  - “Muy caro” → “Costo de oportunidad: ROAS cayendo 15–25%.”

IA Bulk Documentos
- Discovery
  - Volumen de propuestas/briefs/SOPs por semana; cuellos de botella
  - Quién aprueba; requisitos de compliance/estilo
- Cierre
  - “Piloto 2 semanas: 6–12 docs reales + governance. ¿Quieres que prioricemos propuestas o SOPs?”
- Objeciones
  - “Cada cliente es distinto” → “Estructura constante, contenido personalizado.”
  - “Perdemos control” → “Guardrails + versión aprobada + trazabilidad.”
