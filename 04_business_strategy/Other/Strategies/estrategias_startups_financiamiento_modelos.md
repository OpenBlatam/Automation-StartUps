### **Modelo Financiero Base (SaaS B2B)**

#### **1. Revenue Model**

**Monthly Recurring Revenue (MRR):**
```
MRR = Suma de todas las suscripciones mensuales activas
```

**Annual Recurring Revenue (ARR):**
```
ARR = MRR √ó 12
```

**New MRR:**
```
New MRR = New Customers √ó Average Revenue Per User (ARPU)
```

**Expansion MRR:**
```
Expansion MRR = Upsells + Cross-sells + Add-ons
```

**Churned MRR:**
```
Churned MRR = Lost Customers √ó ARPU
```

**Net New MRR:**
```
Net New MRR = New MRR + Expansion MRR - Churned MRR
```

**MRR Growth Rate:**
```
MRR Growth Rate = (MRR este mes - MRR mes anterior) / MRR mes anterior
```

---

#### **2. Customer Metrics**

**Customer Acquisition Cost (CAC):**
```
CAC = Total Sales & Marketing Spend / New Customers Acquired
```

**Lifetime Value (LTV):**
```
LTV = ARPU √ó Gross Margin % √ó (1 / Monthly Churn Rate)
```

**LTV/CAC Ratio:**
```
LTV/CAC = LTV / CAC
Objetivo: >3:1 (m√≠nimo), >5:1 (bueno), >10:1 (excelente)
```

**CAC Payback Period:**
```
CAC Payback = CAC / (ARPU √ó Gross Margin %)
Objetivo: <12 meses
```

**Magic Number:**
```
Magic Number = (Net New ARR este trimestre) / (Sales & Marketing Spend trimestre anterior)
Objetivo: >0.75
```

---

#### **3. Unit Economics**

**Gross Margin:**
```
Gross Margin % = (Revenue - Cost of Goods Sold) / Revenue
Objetivo SaaS: >80%
```

**Contribution Margin:**
```
Contribution Margin = Revenue - Variable Costs
Contribution Margin % = Contribution Margin / Revenue
Objetivo: >70%
```

**Rule of 40:**
```
Rule of 40 = Growth Rate % + Profit Margin %
Objetivo: >40%
```
