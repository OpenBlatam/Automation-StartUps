GUÍA DE IMPLEMENTACIÓN
================================================================================

PASO 1: IDENTIFICAR OBJETIVO
- ¿Qué quieres lograr?
- ¿Qué emoción quieres generar?
- ¿Qué acción quieres que tomen?

PASO 2: SELECCIONAR TÉCNICAS
- Prueba social para credibilidad
- Urgencia para acción inmediata
- Autoridad para confianza
- Contraste para claridad

PASO 3: ESTRUCTURAR TESTIMONIO
- Hook emocional
- Situación inicial
- Solución
- Resultados específicos
- Llamada a la acción

PASO 4: OPTIMIZAR PARA CANAL
- Adaptar longitud
- Ajustar formato
- Incluir elementos visuales
- Agregar CTA apropiado

PASO 5: MEDIR Y OPTIMIZAR
- Trackear conversiones
- A/B test diferentes versiones
- Optimizar basado en datos
- Iterar constantemente
