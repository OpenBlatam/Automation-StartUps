## üìã How to Use This Library

Replace the placeholders in brackets `{ }` with your specific information:
- `{topic}` - Your content topic
- `{audience}` - Your target audience
- `{platform}` - Social media platform
- `{industry}` - Your industry

## üí° Content Ideation Prompts

### General Ideation
```
Generate 20 content ideas for {topic} that would resonate with {audience}. 
Include a mix of educational, entertaining, and promotional content. 
For each idea, suggest the best platform and format.
```

### Audience-Specific Ideation
```
I'm creating content for {audience} in the {industry} industry. 
What are the top 15 questions they ask about {topic}? 
Create content ideas that answer these questions in engaging ways.
```

### Trend-Based Ideation
```
What are the current trending topics related to {topic} in {industry}? 
Generate 10 content ideas that leverage these trends while providing 
value to {audience}.
```

### Problem-Solving Ideation
```
What are the biggest pain points {audience} faces regarding {topic}? 
Create 12 content ideas that solve these problems, including 
different formats (blog posts, videos, infographics, etc.).
```

### Competitor Analysis Ideation
```
Analyze the content strategy of top competitors in {industry}. 
Identify content gaps and opportunities. Generate 15 unique content 
ideas that would differentiate our brand and appeal to {audience}.
```

### Blog Post Creation
```
Write a comprehensive blog post about {topic} for {audience}. 
The post should be {word count} words, include:
- An engaging headline
- Introduction that hooks the reader
- {number} main sections with subheadings
- Practical examples and actionable tips
- A compelling conclusion with a call-to-action
- SEO-optimized meta description
```

### Social Media Post Creation
```
Create {number} {platform} posts about {topic} for {audience}. 
Each post should:
- Be optimized for {platform}'s best practices
- Include relevant hashtags
- Have a clear call-to-action
- Match {tone} tone
- Be between {character count} characters
```

### LinkedIn Post Creation
```
Write a professional LinkedIn post about {topic} for {audience}. 
The post should:
- Be 1,000-1,500 words
- Include personal insights or experiences
- Provide value to {industry} professionals
- Encourage engagement and discussion
- Use a professional but approachable tone
```

### Twitter/X Thread Creation
```
Create a Twitter thread about {topic} for {audience}. The thread should:
- Have {number} tweets
- Start with a hook
- Build on each previous tweet
- Include data or examples
- End with a call-to-action
- Use relevant hashtags
```

### Instagram Caption Creation

Write an Instagram caption for a post about {topic} targeting {audience}. 
The caption should:

## üìÖ Content Calendar Planning Prompts

### Monthly Calendar Creation
```
Create a 30-day content calendar for {platform} focusing on {topic}. 
The calendar should:
- Include daily content ideas
- Balance educational, engagement, and promotional content
- Consider best posting times for {platform}
- Include seasonal relevance for {month}
- Target {audience}
- Align with {goal}
```

### Weekly Content Plan
```
Plan a week of content for {platform} about {topic} for {audience}. 
Create:
- 7 content ideas (one per day)
- Suggested posting times
- Content formats for each day
- Hashtags for each post
- Engagement strategies
```

### Content Pillar Development
```
Develop 5 content pillars for {brand/industry} targeting {audience}. 
For each pillar:
- Define the core theme
- Suggest 10 content ideas
- Recommend best platforms
- Identify content formats
- Suggest posting frequency
```

### Campaign Calendar
```

promoting {product/service/event}. The campaign should:
- Build anticipation before launch
- Create excitement during launch
- Maintain engagement after launch
- Target {audience}
- Use {platforms}
- Align with {goal}
```
