# Checklist de Implementación de Automatizaciones

Usa esta lista para cada automatización. Duplica la sección por item.

## 1) Definición
- [ ] Problema/Oportunidad definida (evidencia y KPI base)
- [ ] Objetivo y meta numérica
- [ ] Dueño y stakeholders
- [ ] Alcance y criterios fuera de alcance

## 2) Diseño
- [ ] Diagrama simple del flujo (trigger → acciones)
- [ ] Reglas y segmentación definidas
- [ ] Fuentes de datos y permisos
- [ ] Mensajes/plantillas aprobados (si aplica)

## 3) Configuración
- [ ] Entorno de prueba creado
- [ ] Variables/secretos seguros
- [ ] Conexiones e integraciones validadas
- [ ] Logs y monitoreo configurados

## 4) Pruebas
- [ ] Casos positivos y negativos
- [ ] Datos de prueba y resultados esperados
- [ ] Revisión por pares
- [ ] Aprobación para producción

## 5) Despliegue
- [ ] Ventana de despliegue definida
- [ ] Plan de rollback
- [ ] Alertas y dashboards activos
- [ ] Comunicación a equipo impactado

## 6) Seguimiento
- [ ] Medición del KPI a 24h / 7d / 30d
- [ ] Documentación actualizada (conclusiones)
- [ ] Iteraciones planificadas (siguiente versión)
