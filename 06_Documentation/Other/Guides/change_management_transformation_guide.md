### **Change Management Models**

#### **Kotter's 8-Step Process**
1. **Create Urgency**: Create a sense of urgency
2. **Form Coalition**: Form a powerful coalition
3. **Create Vision**: Create a vision for change
4. **Communicate Vision**: Communicate the vision
5. **Remove Obstacles**: Remove obstacles to change
6. **Create Short-term Wins**: Create short-term wins
7. **Build on Change**: Build on the change
8. **Anchor Changes**: Anchor changes in culture

#### **ADKAR Model**
- **Awareness**: Create awareness of the need for change
- **Desire**: Create desire to participate in change
- **Knowledge**: Provide knowledge on how to change
- **Ability**: Develop ability to implement change
- **Reinforcement**: Reinforce change to sustain it

#### **Lewin's Change Model**
- **Unfreeze**: Unfreeze current state
- **Change**: Implement change
- **Refreeze**: Refreeze new state
