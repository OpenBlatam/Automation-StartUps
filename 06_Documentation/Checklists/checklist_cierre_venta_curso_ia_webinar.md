# Checklist de Cierre de Venta B2B: Curso de IA y Webinars

**Documento:** Checklist Comercial  
**Producto:** Curso de Inteligencia Artificial + Webinars  
**Versión:** 1.0  
**Fecha:** 2024

---

## VISIÓN GENERAL DEL PROCESO

Este checklist garantiza un proceso sistemático y completo de cierre de venta para el Curso de IA y Webinars, cubriendo desde la calificación inicial hasta la transición exitosa al servicio.

---

## FASE 1: CALIFICACIÓN DEL LEAD

### 1.1 Información Básica del Prospecto
- [ ] **Empresa/Demografía**
  - [ ] Nombre de la empresa completado
  - [ ] Industria/sector identificado
  - [ ] Tamaño de la empresa (micro/pequeña/mediana/grande)
  - [ ] Ubicación geográfica registrada
  - [ ] Sitio web revisado
  - [ ] Presencia en LinkedIn analizada

- [ ] **Información del Contacto**
  - [ ] Nombre completo del decisor
  - [ ] Título/cargo confirmado
  - [ ] Correo electrónico y teléfono verificados
  - [ ] LinkedIn del contacto revisado
  - [ ] Nivel de influencia en la decisión (decisor/influenciador/usuario)

### 1.2 Calificación BANT (Budget, Authority, Need, Timeline)
- [ ] **Budget (Presupuesto)**
  - [ ] Presupuesto disponible confirmado para capacitación/desarrollo
  - [ ] Rango de presupuesto identificado:
    - [ ] Alto valor: $50,000 - $200,000+ USD
    - [ ] Valor mediano: $10,000 - $50,000 USD
    - [ ] Valor bajo: < $10,000 USD
  - [ ] Proceso de aprobación presupuestaria comprendido
  - [ ] Temporada fiscal/año presupuestario conocido

- [ ] **Authority (Autoridad)**
  - [ ] Contacto identificado como decisor final o con acceso al decisor
  - [ ] Comité de decisión identificado (si aplica)
  - [ ] Stakeholders clave mapeados
  - [ ] Proceso de aprobación interno entendido
  - [ ] Nivel jerárquico del contacto documentado

- [ ] **Need (Necesidad)**
  - [ ] Necesidad de capacitación en IA identificada y confirmada
  - [ ] Casos de uso específicos documentados:
    - [ ] Automatización de procesos
    - [ ] Mejora de productividad
    - [ ] Transformación digital
    - [ ] Desarrollo de capacidades internas
    - [ ] Otros: _______________
  - [ ] Número de participantes estimado
  - [ ] Nivel de conocimiento actual del equipo evaluado
  - [ ] Objetivos de aprendizaje definidos

- [ ] **Timeline (Cronograma)**
  - [ ] Fecha objetivo de inicio del curso confirmada
  - [ ] Urgencia del proyecto evaluada
  - [ ] Eventos/deadlines externos que afectan el timing identificados
  - [ ] Ventana de decisión estimada documentada

### 1.3 Evaluación de Valor del Negocio
- [ ] **Criterios para Valor ALTO ($50,000+ USD)**
  - [ ] 50+ participantes confirmados
  - [ ] Contrato multi-año o múltiples cohortes
  - [ ] Personalización significativa requerida
  - [ ] Paquete premium con webinars exclusivos
  - [ ] Certificación empresarial incluida
  - [ ] Integración con programas internos de desarrollo

- [ ] **Criterios para Valor MEDIANO ($10,000 - $50,000 USD)**
  - [ ] 10-50 participantes
  - [ ] Curso estándar con webinars grupales
  - [ ] Personalización moderada
  - [ ] Un ciclo de formación completo
  - [ ] Materiales premium incluidos

- [ ] **Scorecard de Calificación**
  - [ ] Score BANT calculado (mínimo 8/10 para avanzar)
  - [ ] Fit con nuestra oferta evaluado
  - [ ] Potencial de expansión identificado
  - [ ] Decisión: Avanzar / Calificar más / Descartar

### 1.4 Discovery Call Completo
- [ ] Llamada de descubrimiento programada (45-60 min)
- [ ] Preguntas estratégicas planteadas:
  - [ ] ¿Qué desafíos específicos están enfrentando?
  - [ ] ¿Cómo ven el rol de la IA en su organización?
  - [ ] ¿Qué éxito se vería como en 6-12 meses?
  - [ ] ¿Qué ha intentado antes y qué funcionó/no funcionó?
- [ ] Grabación y transcripción de la llamada (con permiso)
- [ ] Notas clave documentadas en CRM
- [ ] Next steps acordados con el prospecto

---

## FASE 2: PRESENTACIÓN DE PROPUESTA

### 2.1 Preparación de la Propuesta
- [ ] **Análisis y Personalización**
  - [ ] Necesidades específicas identificadas en discovery call revisadas
  - [ ] Propuesta personalizada creada según valor del negocio:
    - [ ] Propuesta ALTO valor: PPT ejecutivo + documento detallado + ROI calculator
    - [ ] Propuesta MEDIANO valor: Documento estructurado + casos de éxito
  - [ ] Contenido del curso adaptado a industria/sector
  - [ ] Webinars relevantes seleccionados según intereses

- [ ] **Materiales de Propuesta**
  - [ ] Propuesta comercial formateada profesionalmente
  - [ ] Programa detallado del curso incluido
  - [ ] Calendario de webinars propuesto
  - [ ] Biografía de instructores incluida
  - [ ] Casos de éxito relevantes documentados
  - [ ] Testimonios de clientes similares incluidos
  - [ ] ROI calculator / business case preparado
  - [ ] Comparativa con alternativas (si aplica)

### 2.2 Estructura de la Propuesta por Valor

**Para NEGOCIOS DE ALTO VALOR:**
- [ ] Deck ejecutivo (15-20 slides)
  - [ ] Situación actual y desafíos
  - [ ] Solución propuesta personalizada
  - [ ] Roadmap de implementación
  - [ ] ROI proyectado con métricas específicas
  - [ ] Plan de capacitación por fases
  - [ ] Timeline detallado con milestones
- [ ] Documento técnico complementario (30+ páginas)
- [ ] Propuesta de webinars exclusivos para la organización
- [ ] Opciones de certificación empresarial
- [ ] Plan de seguimiento y medición de resultados

**Para NEGOCIOS DE VALOR MEDIANO:**
- [ ] Propuesta comercial estructurada (10-15 páginas)
  - [ ] Resumen ejecutivo
  - [ ] Programa del curso
  - [ ] Webinars incluidos
  - [ ] Pricing y términos
  - [ ] Timeline
- [ ] Casos de éxito relevantes
- [ ] FAQ documentado

### 2.3 Pricing y Packaging
- [ ] **Opciones de Pricing Preparadas**
  - [ ] Precio base calculado según número de participantes
  - [ ] Descuentos por volumen definidos (si aplica)
  - [ ] Opciones de pago presentadas:
    - [ ] Pago único con descuento
    - [ ] Pago fraccionado
    - [ ] Pago anual con beneficios
  - [ ] Add-ons identificados:
    - [ ] Webinars adicionales
    - [ ] Sesiones 1:1 de consultoría
    - [ ] Materiales adicionales
    - [ ] Certificaciones premium

- [ ] **Negociación Room Preparada**
  - [ ] Precio mínimo viable calculado
  - [ ] Concesiones posibles identificadas
  - [ ] Condiciones de descuento documentadas

### 2.4 Presentación de la Propuesta
- [ ] Reunión de presentación programada:
  - [ ] Fecha y hora confirmada
  - [ ] Duración: 60-90 min para alto valor, 45-60 min para mediano
  - [ ] Todos los stakeholders clave invitados
  - [ ] Confirmación de asistencia recibida
- [ ] Preparación del equipo:
  - [ ] Presentador principal asignado
  - [ ] Soporte técnico disponible si es necesario
  - [ ] Demo preparada (si aplica)
  - [ ] Materiales de respaldo listos
- [ ] Tecnología verificada:
  - [ ] Zoom/Teams configurado y probado
  - [ ] Compartir pantalla funcionando
  - [ ] Audio y video verificados
- [ ] Agenda de la reunión compartida previamente

### 2.5 Durante la Presentación
- [ ] Introducción y contexto (5 min)
- [ ] Presentación de necesidades identificadas (10 min)
- [ ] Solución propuesta - descripción del curso (20 min)
- [ ] Componente de webinars explicado (15 min)
- [ ] ROI y beneficios cuantificables (10 min)
- [ ] Pricing y packaging (10 min)
- [ ] Q&A y objeciones manejadas (20 min)
- [ ] Next steps acordados (5 min)

### 2.6 Post-Presentación
- [ ] Propuesta formal enviada por email en 24 horas
- [ ] Slides de presentación compartidas
- [ ] Materiales adicionales enviados según solicitudes
- [ ] Seguimiento programado en CRM:
  - [ ] Día 1: Email de agradecimiento
  - [ ] Día 3: Email con información adicional si solicitada
  - [ ] Día 5: Llamada de seguimiento

## FASE 3: NEGOCIACIÓN

### 3.1 Manejo de Objeciones Comunes
- [ ] **Objeción: Presupuesto**
  - [ ] Justificación de ROI presentada
  - [ ] Opciones de flexibilización de pago ofrecidas
  - [ ] Comparativa con costo de no actuar presentada
  - [ ] Valor incrementado mostrado (webinars exclusivos, etc.)
  - [ ] Alternativas de scope discutidas

- [ ] **Objeción: Timing**
  - [ ] Fechas alternativas propuestas
  - [ ] Opción de inicio escalonado discutida
  - [ ] Urgencia estratégica reforzada
  - [ ] Ventanas de oportunidad identificadas

- [ ] **Objeción: Competencia / Alternativas**
  - [ ] Diferenciadores clave destacados
  - [ ] Casos de éxito específicos compartidos
  - [ ] Comparativa detallada preparada
  - [ ] Beneficios únicos enfatizados (webinars, certificación, etc.)

- [ ] **Objeción: Necesidad / Prioridad**
  - [ ] Impacto del status quo visualizado
  - [ ] Oportunidad de transformación destacada
  - [ ] Riesgo competitivo si no actúan mencionado
  - [ ] Beneficios inmediatos y a largo plazo enfatizados

- [ ] **Objeción: Confianza / Credibilidad**
  - [ ] Credenciales de instructores compartidas
  - [ ] Testimonios y casos de éxito presentados
  - [ ] Certificaciones y reconocimientos mencionados
  - [ ] Garantías o políticas de satisfacción explicadas

### 3.2 Estrategia de Negociación por Valor

**ALTO VALOR:**
- [ ] Concesiones preparadas:
  - [ ] Descuento del 10-15% con condiciones (pago anticipado, multi-año)
  - [ ] Webinars adicionales sin costo
  - [ ] Sesiones de consultoría incluidas
  - [ ] Certificación empresarial premium
  - [ ] Timeline extendido si necesario
- [ ] Tácticas:
  - [ ] Enfoque en ROI y resultados, no solo precio
  - [ ] Paquetes de valor agregado
  - [ ] Establecer relación a largo plazo
  - [ ] Involucrar C-level si es necesario

**VALOR MEDIANO:**
- [ ] Concesiones preparadas:
  - [ ] Descuento del 5-10% con pago anticipado
  - [ ] Acceso extendido a materiales
  - [ ] Webinar adicional incluido
  - [ ] Timeline flexible
- [ ] Tácticas:
  - [ ] Enfoque en valor práctico y resultados
  - [ ] Facilitar el proceso de aprobación
  - [ ] Ofrecer pruebas o pilots si aplica

### 3.3 Negociación de Términos
- [ ] **Precio y Pagos**
  - [ ] Precio final acordado
  - [ ] Estructura de pago negociada:
    - [ ] % inicial al firmar
    - [ ] % al inicio del curso
    - [ ] % a la finalización
  - [ ] Términos de pago documentados (NET 30, etc.)

- [ ] **Scope y Contenido**
  - [ ] Número exacto de participantes confirmado
  - [ ] Módulos del curso confirmados
  - [ ] Webinars incluidos especificados
  - [ ] Personalización acordada y documentada
  - [ ] Materiales adicionales confirmados

- [ ] **Timeline y Fechas**
  - [ ] Fecha de inicio confirmada
  - [ ] Cronograma de webinars acordado
  - [ ] Fechas de milestones definidas
  - [ ] Duración total del programa confirmada

- [ ] **Condiciones Especiales**
  - [ ] Requisitos técnicos identificados
  - [ ] Idioma de instrucción confirmado
  - [ ] Horarios preferidos acordados
  - [ ] Cualquier requerimiento especial documentado

### 3.4 Cierre de la Negociación
- [ ] Todos los términos acordados verbalmente
- [ ] Resumen de acuerdos enviado por email
- [ ] Confirmación de aceptación recibida del cliente
- [ ] Siguiente paso: preparación de contrato legal

## FASE 4: CIERRE LEGAL

### 4.1 Preparación del Contrato
- [ ] **Documentos Requeridos**
  - [ ] Contrato de servicios preparado según tipo de negocio:
    - [ ] Contrato Enterprise (alto valor)
    - [ ] Contrato Estándar (valor mediano)
  - [ ] Acuerdo de NDA (si aplica)
  - [ ] Términos y condiciones revisados
  - [ ] Política de cancelación/devolución documentada

- [ ] **Contenido del Contrato**
  - [ ] Información de las partes completa
  - [ ] Descripción detallada de servicios:
    - [ ] Curso especificado con módulos
    - [ ] Webinars listados con fechas/temas
    - [ ] Materiales incluidos
    - [ ] Certificaciones (si aplica)
  - [ ] Precio y términos de pago claramente establecidos
  - [ ] Timeline y fechas críticas
  - [ ] Obligaciones de ambas partes
  - [ ] Política de cancelación y reembolsos
  - [ ] Propiedad intelectual definida
  - [ ] Confidencialidad y NDA
  - [ ] Limitación de responsabilidad
  - [ ] Resolución de disputas
  - [ ] Jurisdicción y leyes aplicables

### 4.2 Revisión Legal
- [ ] Contrato revisado por departamento legal interno
- [ ] Términos estándar aplicados
- [ ] Cláusulas especiales añadidas según necesidad
- [ ] Cumplimiento con regulaciones verificada
- [ ] Versión final aprobada para envío

### 4.3 Envío y Negociación del Contrato
- [ ] Contrato enviado al cliente con:
  - [ ] Carta de presentación
  - [ ] Instrucciones de firma
  - [ ] Fecha límite para firma establecida (7-14 días)
  - [ ] Preguntas frecuentes sobre términos legales
- [ ] Sistema de seguimiento activado en CRM
- [ ] Recordatorios programados si no se recibe respuesta

### 4.4 Negociación de Términos Legales (si aplica)
- [ ] **Términos Comúnmente Negociados**
  - [ ] Cláusulas de cancelación
  - [ ] Términos de pago
  - [ ] Propiedad intelectual de materiales personalizados
  - [ ] SLA (Service Level Agreements) si aplica
  - [ ] Penalizaciones por cancelación

- [ ] **Proceso de Negociación**
  - [ ] Cambios solicitados por cliente recibidos
  - [ ] Evaluación de cambios con equipo legal
  - [ ] Respuesta a cambios proporcionada:
    - [ ] Aceptados
    - [ ] Modificados
    - [ ] Rechazados con justificación
  - [ ] Rondas de negociación documentadas
  - [ ] Versión final acordada

### 4.5 Firma del Contrato
- [ ] Contrato firmado por ambas partes recibido
- [ ] Versiones firmadas almacenadas:
  - [ ] Sistema de gestión de documentos
  - [ ] CRM actualizado con estado "Contrato Firmado"
  - [ ] Backup en ubicación segura
- [ ] Confirmación de firma enviada al cliente
- [ ] Implementación de pagos iniciada según términos

### 4.6 Post-Firma Legal
- [ ] Factura inicial generada (si aplica)
- [ ] Información del cliente para onboarding compartida con equipos internos
- [ ] Transición a fase de servicio iniciada

## NOTAS ADICIONALES Y MEJORES PRÁCTICAS

### Diferenciadores Clave del Curso de IA y Webinars
- Combinación única de curso estructurado + webinars interactivos
- Instructores con experiencia práctica en implementación empresarial
- Certificación reconocida en la industria
- Comunidad de alumnos para networking continuo
- Actualización constante de contenido según evolución de IA

### Señales de Alerta (Red Flags)
- Prospecto no tiene presupuesto confirmado después de 2 reuniones
- No hay urgencia real o necesidad clara
- Contacto no tiene autoridad para decidir y no introduce al decisor
- Expectativas desalineadas con nuestra oferta
- Comparaciones constantes con opciones mucho más baratas sin reconocer valor

### Tácticas de Cierre Efectivas
- **Urgencia legítima**: Fechas limitadas, promociones con plazo
- **Escasez**: Cupos limitados en cohorte
- **Prueba social**: Casos de éxito de empresas similares
- **Garantía**: Política de satisfacción o reembolso
- **Próximo paso claro**: "¿Qué necesitamos para avanzar?"
