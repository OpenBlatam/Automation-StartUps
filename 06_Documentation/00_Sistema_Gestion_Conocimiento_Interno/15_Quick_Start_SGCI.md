# ⚡ Quick Start SGCI (30 minutos)

## 1) Entiende el sistema (10 min)
- Lee `00_MANUAL_MAESTRO_SGCI.md`
- Revisa la taxonomía y metadatos (07_*)

## 2) Documenta un proceso (10 min)
- Usa `01_Guia_Documentacion_Procesos.md`
- Crea borrador con plantilla de Proceso

## 3) SOP y control (5 min)
- Deriva 1 SOP clave con `02_Guia_Actualizacion_SOPs.md`
- Define 2-3 KPIs y controles

## 4) Publica y comparte (5 min)
- Guarda en ruta canónica
- Abre PR y solicita revisión

Checklist
- Metadatos completos, enlaces verificados, próxima revisión fijada
