#### Área de Compras

- Reducción objetivo: 25%

- Inversión año 1: $25,000
- Inversión año 2: $10,000

**Cálculos Automáticos:**
```
Ahorro Año 1: $2,000,000 × 25% = $500,000
Inversión Total: $85,000 + $25,000 + $10,000 = $120,000
ROI Año 1: ($500,000 - $120,000) / $120,000 × 100 = 317%
Payback Period: $120,000 / $500,000 × 12 = 2.9 meses
NPV (10%): $500,000 / 1.1 + $500,000 / 1.1² + $500,000 / 1.1³ - $120,000 = $1,240,000
```

## CALCULADORA DE MÉTRICAS FINANCIERAS

### Métricas de Rentabilidad

#### ROI (Return on Investment)
**Fórmula:**
```
ROI = (Ganancia - Inversión) / Inversión × 100
```

**Ejemplo:**
```
ROI = ($2,823,000 - $850,000) / $850,000 × 100 = 232%
```

**Herramienta Interactiva:**
- [ ] Calculadora de ROI
- [ ] Gráfico de ROI por año
- [ ] Comparación de ROI por área
- [ ] Proyección de ROI futuro

#### NPV (Net Present Value)
**Fórmula:**
```
NPV = Σ(Flujo de Caja / (1 + r)^t) - Inversión Inicial
```

**Ejemplo:**
```
NPV = $2,823,000/1.1 + $2,950,000/1.1² + $3,200,000/1.1³ - $850,000 = $7,450,000
```

**Herramienta Interactiva:**
- [ ] Calculadora de NPV
- [ ] Gráfico de NPV por año
- [ ] Análisis de sensibilidad a tasa de descuento
- [ ] Comparación de NPV por área

#### IRR (Internal Rate of Return)
**Fórmula:**
```
NPV = 0 = Σ(Flujo de Caja / (1 + IRR)^t) - Inversión Inicial
```

**Ejemplo:**
```
IRR = 340%
```

**Herramienta Interactiva:**
- [ ] Calculadora de IRR
- [ ] Gráfico de IRR por año
- [ ] Comparación de IRR por área
- [ ] Análisis de sensibilidad

#### Payback Period
**Fórmula:**
```
Payback Period = Inversión Inicial / Flujo de Caja Anual
```

**Ejemplo:**
```
Payback Period = $850,000 / $2,823,000 × 12 = 3.6 meses
```

**Herramienta Interactiva:**
- [ ] Calculadora de Payback Period
- [ ] Gráfico de Payback Period por área

- [ ] Comparación con benchmarks
